> 研究来源：网易官方战斗机制讲解、bilibili WIKI（星际猎人-无尽的拉格朗日）、官方维护公告、玩家实测攻略。

## 0. 总览：一场战斗由哪些子系统构成

一场舰队交战（小编队 vs 小编队）涉及以下并行运作的子系统：

```
┌─────────────────────────────────────────────────────────┐
│  战场（阵型：敌方前中后排 / 交战区 / 我方前中后排）         │
├─────────────────────────────────────────────────────────┤
│  我方舰队                                                  │
│   ├─ 多艘舰船，每艘位于某一排（前/中/后）                   │
│   │   └─ 舰船本体：结构值(HP)、抵抗(实弹减伤)、护盾(能量减伤)│
│   │       速度、闪避率                                     │
│   │   └─ 多个武器系统（各自独立运作）                       │
│   │       ├─ 攻击序列（目标优先级 AI）                     │
│   │       ├─ 攻击循环（单发×次数/持续+冷却）                │
│   │       ├─ 伤害类型（实弹/能量）+ 命中率（按目标类型）     │
│   │       ├─ 武器自身结构值（可被摧毁→失能）               │
│   │       └─ 特殊：反导拦截 / 防空                         │
│   ├─ 舰载机（战机/护航艇，由载机舰搭载，空中作战）          │
│   └─ 防空体系（主动防空/防空支援/反击防空）                │
└─────────────────────────────────────────────────────────┘
```

**仿真核心 = 离散事件仿真（DES）**：每个武器系统是一个独立"定时器"，按各自攻击循环周期触发攻击事件，事件经过"目标选择→拦截判定→命中判定→伤害结算→子系统摧毁判定"链路。

## 1. 伤害的二分法（最核心机制）

游戏**只有两种伤害类型**，对应两种独立防御。这是伤害公式的第一性原理。

| 伤害类型 | 包含武器 | 对应防御 | 结算方式 |
|---------|---------|---------|---------|
| **实弹伤害** | 火炮、轨道炮、导弹、鱼雷、机载机炮等 | **抵抗伤害**（数值，标准=10） | 需"破防"，单发伤害受抵抗削弱 |
| **能量伤害** | 脉冲武器、离子炮、等离子体 | **护盾值**（百分比，如2.0%） | 伤害×(1−护盾值) |

### 1.1 基础伤害公式
```
实弹：实际伤害 = 单发伤害 × 实弹破防系数(单发伤害 vs 抵抗值)
能量：实际伤害 = 单发伤害 × (100% − 护盾值)
```
> 来源：4399《战舰入门攻略》；官方战斗机制讲解（能量武器"能有效的穿透传统装甲"=忽略抵抗，只吃护盾）

### 1.2 实弹"破防"机制（破防判定）
实弹伤害面对抵抗值时存在**破防阈值**（玩家攻略称"破防"）：
- 单发伤害显著高于抵抗值 → 接近全额伤害
- 单发伤害接近或低于抵抗值 → 伤害被大幅削减（多次小额伤害武器打高甲目标极其吃亏）

> 这是"高单发武器 vs 高频低伤武器"分化的根源。官方2023/4/6维护公告明确：**面板"对舰火力"显示为打击标准护甲(抵抗=10)目标时的火力值**，即面板DPM已假设目标抵抗=10。

### 1.3 结构值（HP）
舰船被摧毁的总血量。**注意区分两层结构值**：
- **舰船本体结构值**：归零→舰船沉没
- **武器系统结构值**：每个武器子系统有独立HP，归零→该武器永久失能（见§7）

## 2. 武器攻击循环机制（DPS 来源）

武器不是"每帧攻击"，而是按**攻击循环（Attack Cycle）**周期性开火。

### 2.1 循环参数（每个武器系统都有）
| 参数 | 含义 | 示例 |
|------|------|------|
| 单发伤害 (DPH) | 每发炮弹伤害 | 75 |
| 攻击次数 | 一个循环内开火次数 | 4 |
| 攻击持续时间 | 连续开火阶段时长(秒) | 20 |
| 冷却时间 | 开火后停火时长(秒) | 15 |

### 2.2 DPS / DPM 公式
```
单循环总伤害 = 单发伤害 × 攻击次数
单循环时长   = 攻击持续时间 + 冷却时间
DPS = (单发伤害 × 攻击次数) / (攻击持续时间 + 冷却时间)
DPM = DPS × 60     ← 游戏面板常用单位（对舰/对空/攻城火力）
```
> 来源：TapTap《武器火力机制解析》、官方战斗机制讲解"武器攻击循环"

### 2.3 强化对循环的影响
- **冷却缩减**：减少"冷却时间"（点满约 −15%）；某些策略技能可大幅缩短（如 −40%、−80%）
- **伤害加成**：直接乘到单发伤害（点满约 +10%；特种弹药等词条额外加）
- **暴击**：见 §4

## 3. 命中率机制（复刻战报的关键）

### 3.1 命中率极限
```
命中率最低 10%，最高 95%（不存在 0% 和 100% 命中）
```
> 来源：WIKI《武器命中率与拦截率计算公式》（2025-12-05，作者北极护航队等）

### 3.2 对舰命中率公式
```
对舰实际命中率 = 基础命中率 × (1 + 强化命中加成 − 对方闪避率 − 自身策略/对方造成的命中减益)
```
- **基础命中率**：武器对**每种目标类型**有不同基础值（见§3.4）
- **强化命中加成**：蓝图加点、信息链（枪防/光防）、策略技能（如"精准制导"+40%）
- **对方闪避率**：见 §5
- 命中减益：阋神星"火力倾泻"自降命中、AT021攻击机干扰等

### 3.3 对空命中率公式（防空武器打舰载机）
```
对空实际命中率 = 基础命中率 × 防空协同效率 × (1 + 强化命中加成 − 对方闪避率 − 命中减益 + 己方命中增益)
```
- **防空协同效率**：随武器安装数递减（见§8.2）
- 己方命中增益：仅枪防/光防的"信息链"

### 3.4 基础命中率按目标类型分别计算
武器对**不同目标类型**有独立基础命中率，需分别记录。例：
- 火炮 vs 护卫舰/驱逐舰/巡洋舰/战机/护航艇 各有不同基础命中
- 反小武器对小船高命中，对大船低命中；反大武器反之

> **面板命中率为区间**（如驱逐舰主武器对护卫舰 50%-70%），引擎建模为 `{min, max}`，
> 每发开火在区间内独立 roll 一个 base 值，再套 §3.2 公式做命中判定。
> 已标定的精确值可用单值（等价于 `min==max` 的退化区间），向后兼容。

> 仿真数据模型：每个武器系统存一个 `baseHit{护卫,驱逐,巡洋,战巡,航母,战机,护航艇,建筑}` 映射，
> 每个取值为单值或区间。

## 4. 暴击机制

```
暴击触发：基础约 15% 暴击概率（可被蓝图/策略提升至 50%）
暴击伤害：暴击伤害倍率（攻略提及 40%~100% 额外伤害；如A2系统暴击伤害+100%）
```
- 暴击为独立概率判定，命中后单独 roll
- 暴击率/暴击伤害均可通过蓝图强化项和策略提升
- 部分舰船有特殊暴击条件（如目标结构值<50%时暴击伤害+100%）

> 来源：官方2021/10维护公告（"攻击有15%概率额外造成暴击伤害"）；官方2023/4/6公告（暴击伤害提高100%）

## 5. 闪避 / 规避机制

### 5.1 闪避率
```
闪避率由动力系统决定，作为"对方命中率"的减项（见§3.2、§3.3）
```
- 闪避点满通常可显著降低被命中率
- 制导武器有独立的"制导武器闪避率"加点项

### 5.2 特殊规避策略（条件触发）
- **机动规避**（阋神星等）：舰船本体结构值 < 20% 时，闪避 +40%，持续40秒，每场触发一次
- 此类为"舰船策略技能"，需在仿真中作为状态机条件处理

> 来源：红手指《阋神星攻略》；WIKI命中率公式页

## 6. 攻击序列（目标选择 AI）

### 6.1 武器目标选择三大类
| 序列类型 | 主要目标 | 典型 |
|---------|---------|------|
| **反小型（反小）** | 护卫舰/驱逐舰/战机 | 速射炮、阋神星156炮 |
| **反大型（反大）** | 巡洋/战巡/航母/超主力舰 | 重炮、刺水母无人机 |
| **防空** | 空中目标（战机/护航艇） | 拦截机、防空炮 |
| **攻城** | 建筑 | 攻城鱼雷、轰炸机 |

### 6.2 序列优先级规则
1. 武器有**固定目标优先级排序**（如速射炮：护卫→战机→驱逐→巡防）
2. **策略技能可覆盖**原有序列（如"优先攻击超主力舰"），策略 > 默认序列
3. **跨排攻击**：导弹/鱼雷/舰载机可跨越前中后排直接打后排；直射武器受射程/排限制
4. **旗舰技能**可改变选择规则（如"绝境作战"：优先打结构值最低的4个目标）

### 6.3 反击触发
部分武器在舰船遭受对应类型目标攻击时才优先反击（被攻击触发机制）。

> 来源：7723配舰教学；网易大神《攻击序列解析》；官方2021/10/20维护公告（攻击序列调整案例）

## 7. 武器系统被摧毁机制（动态状态）

- **每个武器子系统有独立结构值**
- 部分舰载机/武器的攻击能对**目标主武器系统**造成额外伤害（"攻击武器系统"词条）
- 武器系统结构值归 0 → **永久失能**，本场战斗不再开火，战后才能紧急维修
- **仿真影响**：必须在每轮伤害结算后检查各子系统存活状态，动态剔除失能武器的攻击事件

> 来源：官方战斗机制讲解（"当主武器系统的结构值降至0，将会陷入无法运作的失能状态…仅能在战斗结束后维修"）

## 8. 战机 / 护航艇 / 防空 / 反导拦截

### 8.1 舰载机分类
| 类别 | 特点 |
|------|------|
| **战机** | 携带少量燃料，出击一段时间后**返航补给**（往返周期），如拦截机/战斗机/攻击机/轰炸机 |
| **护航艇** | 长时间持续在外作战，无需频繁返航 |

舰载机可**跨越阵型直接攻击中后排**，是突破前排肉盾的关键。

### 8.2 防空协同效率（武器安装数衰减，关键公式）
```
武器安装数 ×1 → 100%
            ×2 → 80%
            ×3 → 70%
            ×4 → 60%
            ×5 → 45%
            ×6 → 30%
对空实际命中率 = 基础命中率 × 防空协同效率 × (其他修正...)
```
> 来源：WIKI《对空伤害计算与防空协同效率解析》

### 8.3 对空增伤（隐藏系数）
```
对空增伤比 = 1 + 防空特种弹药值 ÷ 基础单发伤害
对空单发伤害 = 对舰单发伤害 × 对空增伤比
```

### 8.4 防空范围（三种，均为主动防空）
- **主动防空（全场防空）**：覆盖双方全部空域；常见于空优机/拦截机（米斯特拉、砂龙、CV-T800等）
- **防空支援（半场防空）**：覆盖我方前中后排空域（林鸮、孢子、刺水母防空型等）
- **同排防空**：仅覆盖同排空域（光锥防空型、苔原、枪防等）
- **反击防空**：被敌机攻击时反击（无目标时以10%命中率随机攻击）

### 8.5 反导拦截（针对导弹/鱼雷）
```
总拦截率 = 1 − (1−全队拦截率)^a × (1−同排拦截率)^b × (1−自身拦截率)
   a = 队伍中"全队拦截率"舰船数量
   b = 与该拦截船同排的"同排拦截率"舰船数量
实际拦截率 = 总拦截率 × (1 − 敌方反拦截率)

每艘拦截率 = 基础拦截率 × (1 + 拦截率加成 − 反拦截率)
```
- 拦截是**累乘**过程（多艘拦截船依次拦截剩余）
- 拦截是**百分比拦截**，不受对方投射数量影响
- 拦截与防空机制**相互独立**

> 来源：WIKI《武器命中率与拦截率计算公式》

## 9. 阵型与前中后排

### 9.1 三段式阵型
- **前排**：肉盾抗伤（高结构/高抵抗舰船）
- **中排**：主力输出 + 防空
- **后排**：航母/支援舰/远程输出

### 9.2 跨排规则
- **直射武器**（火炮/轨道炮/能量直射）：受射程与排位限制，主要打前排
- **投射武器**（导弹/鱼雷）：可跨排攻击后排，但会被反导拦截
- **舰载机**：完全无视阵型，直接攻击任意排

### 9.3 编队约束
- 舰队受**指挥值上限**约束（如460指挥值）
- 舰船与舰载机有指挥值占用配比

> 来源：游民星空《舰船编队指南》；B站专栏《舰队组建心得》

## 10. 蓝图技术值强化体系

树状加点系统，强化项作用于上述各机制：
- **伤害**：+10%（满）/ 特种弹药额外
- **冷却**：−15%（满）
- **暴击率/暴击伤害**：可堆至 50%
- **命中率**：按目标类型加成
- **闪避**：常规/制导分别加点
- **结构值/抵抗/护盾**：装甲系统
- **速度**：常规/曲率

加点可导出为**加点码**（玩家社区通用，如"篝火社"小程序支持导入导出）。

### 赛季调校 / 永久调校（额外固定加成）
WIKI 舰船页的 `对舰扩展1/2`、`结构扩展1/2` 等字段记录，如：
- 火力倾泻：武器冷却 −5%
- 龙骨结构增强：舰船结构值 +7%
- 炮弹改良：系统内武器伤害 +5%

## 11. 仿真引擎关键决策点（给开发的结论）

| 决策点 | 结论 |
|--------|------|
| 仿真范式 | **离散事件仿真**：武器为独立定时器，事件驱动 |
| 时间步长 | **浮点秒**（事件驱动，非固定 tick）。面板冷却/持续/锁定会出现小数（如5.5s），事件时间直接存浮点，不取整 |
| 随机性 | 命中/暴击/拦截/反击触发均为独立 roll，需固定随机种子以复现战报 |
| 数据最小单元 | "武器系统"（非整船），需逐武器记录：DPH/次数/持续/冷却/伤害类型/基础命中映射/序列/拦截/子系统HP |
| 黑盒部分 | 实弹破防曲线、基础命中率的精确值、暴击基础值——只能近似，需实测校准 |
| 目标选择 | 每个攻击事件独立执行攻击序列逻辑（含跨排/策略覆盖/旗舰技能） |
| 状态机 | 舰船(存活/残血触发规避)、武器系统(存活/失能)、舰载机(出击/返航)、策略技能冷却 |

bilibili WIKI 的 MediaWiki API 完全可用：
- `api.php?action=query&list=allpages` — 列出所有舰船页（已确认 68+ 舰船页）
- `api.php?action=query&prop=revisions&rvprop=content&titles=...` — 获取页面 wikitext

每艘舰船页用**模板参数化**记录结构化数据（已抓取"阋神星I级-重炮驱逐舰"验证）：
```
{{新舰船
|名称=... |攻击属性=实弹/能量 |武器类型=火炮 |武器定位=直射/投射
|舰船站位=前排/中排/后排 |TAG=周期集火,本体拦截,反击防空 |旗舰=舰队集火
}}
{{新舰船/基础数据
|对舰=4963 |对空=720 |攻城=1239      ← DPM 面板值
|结构=30540 |抵抗=20 |护盾=2.0       ← 防御
|巡航=850 |曲率=4250                 ← 速度
|对舰扩展1=... |结构扩展1=...        ← 调校加成
}}
```

**局限**：单舰船页只给**汇总面板DPM**，不直接给逐武器的 DPH/攻击循环/基础命中率。
→ 这些更细的数据散落在"舰船蓝图档案"类页面或需结合游戏内截图/玩家实测补充。
→ MVP 阶段可先用面板 DPM 做整体对舰输出，再逐步细化到武器级。
