## 一、混沌级轨道炮 vs CV3000（2场交叉验证）

### 配置
- 攻击方：2艘康纳马拉混沌级轨道炮巡洋舰(bp=51201)，结构93104/艘
- 防守方：1艘CV3000航母(bp=80201)，结构370192，无闪避

### 两场对比

| 指标 | 无加成(1349s) | 有加成+25%(1083s) | 说明 |
|---|---|---|---|
| 战报时长 | 1349s | 1083s | |
| 击毁时刻 | 1348s | 1082s | HP曲线type=97 |
| 后摇 | 1s | 1s | ★实测(非旧假设5s) |
| 主炮命中 | 394发 | 395发 | 固定(HP/perHit) |
| 主炮命中率p | **61.6%** | **77.1%** | |
| base(p÷1.25) | 61.6% | **61.7%** | ★两场一致 |
| 副炮伤害(UI) | 3772 | 2912 | |
| 混沌受损(第4列) | 62250 | 43755 | CV3000反击 |

### HP曲线格式（已破解）
```
第3列 = CV3000(受击方)结构值，370192→0
第4列 = 混沌(攻击方)总结构值，186208→142453（加成版）
type=97 @击毁时刻 = 精确事件
```

### 结论
- **主炮base=61.6%**（面板50-70%）
- 加成+25%全额生效（p=77.1%=61.7%×1.25），**无衰减**
- 后摇=1s，前摇T0≈7s

## 三、关键参数修正（vs 04文档）

| 参数 | 04文档 | 实测 | 影响 |

| 后摇 | 5s | **1s** | 击毁时刻后移4s，旧反推的命中率略偏低 |
